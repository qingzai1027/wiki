### Math.floor( )

`Math.floor()` 返回小于或等于一个给定数字的最大整数。

```js
举例：
Math.floor( 45.95);
// 45
Math.floor( 45.05);
// 45
Math.floor( 4 );
// 4
Math.floor(-45.05);
// -46
Math.floor(-45.95);
// -46
```



箭头函数 `() => x` 是 `()=> {return x;}` 的有效简写
